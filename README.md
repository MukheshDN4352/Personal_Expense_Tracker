# Personal Expense Tracker (Arthavya)

A full-stack Personal Expense Tracker Web Application built using the MERN stack that helps users manage income, expenses, and custom budget cycles with clear financial insights.

The application allows users to track their financial activities, visualize spending patterns, and stay within budget using an interactive dashboard.

---

## Features

- User Authentication (Signup / Login)
- Dashboard with Total Balance, Income, and Expenses
- Pie Chart for Income vs Expense visualization
- Bar Chart showing last 30 days expense activity
- Recent Transactions history
- Monthly financial summary
- Add and delete Income transactions
- Add and delete Expense transactions
- Download Income and Expense data as Excel files
- Custom Budget Cycle with spending limits
- Budget alerts when nearing or exceeding the limit
- Remaining budget and remaining days tracking
